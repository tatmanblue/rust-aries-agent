# Rust Aries Agent

The primary goal of this project is to create an aries agent compliant with all RFCS and works with other
existing agents such as [ACAPY](https://github.com/hyperledger/aries-cloudagent-python)

The secondary goal of this project will be to create an agency for aries compliant agents compatiable 
with the fist goal stated above.
